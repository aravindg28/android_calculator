
APPLICATION DESCRIPTION:

    The calculator android application is capable of:
      1) Performing arithmetic operations like +,-,*,/
      2) Performing memory operations like M+,M-,MR and MC
      3) Performing toggle operation (+/-)
      4) Performing clear and delete operations
   
NOTE:

    1) The application is designed for Nexus 6 (1440*2560) and phones having similar screen dimension 
    2) The application supports only input length of 10 exlucding decimals and operators.
